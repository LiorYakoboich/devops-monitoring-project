# DevOps Monitoring Project

## Overview
A monitoring system built with FastAPI, Prometheus, and Grafana.

## Tech Stack
- FastAPI
- Docker
- Prometheus
- Grafana
- GitHub Actions

## Features
- Request rate monitoring
- Request latency tracking
- Health check endpoint
- Metrics exposure for Prometheus

## Architecture
Client → FastAPI → Prometheus → Grafana

## Run locally
docker compose up --build
